# Система Учета Заявок на Ремонт Бытовой Техники

Система для управления заявками на ремонт бытовой техники с графическим интерфейсом и разграничением доступа по ролям пользователей.

## Описание проекта

Приложение позволяет:
- Создавать и управлять заявками на ремонт
- Назначать мастеров на заявки
- Отслеживать статусы выполнения заявок
- Генерировать QR-коды для заявок
- Формировать отчеты и статистику
- Управлять пользователями и их ролями

Система поддерживает несколько ролей пользователей: Администратор, Менеджер, Оператор, Мастер и Клиент, каждая с соответствующими правами доступа.

## Стек технологий

- **Python 3.8+** — основной язык программирования
- **PyQt6** — фреймворк для создания графического интерфейса
- **PostgreSQL** — система управления базами данных
- **psycopg2** — адаптер для работы с PostgreSQL
- **qrcode** — библиотека для генерации QR-кодов

## Структура проекта

```
praktika_3/
├── gui.py                    # Главный модуль с графическим интерфейсом
├── main_app.py               # Бизнес-логика приложения
├── database_module.py        # Модуль для работы с базой данных
├── import_data.py            # Скрипт для импорта данных из CSV
├── table.sql                 # SQL-схема базы данных
├── inputDataUsers.csv        # Данные пользователей
├── inputDataRequests.csv     # Данные заявок
├── inputDataComments.csv     # Данные комментариев
```

### Основные модули

- **gui.py** — содержит классы для создания интерфейса (диалоги, главное окно, таблицы)
- **main_app.py** — реализует бизнес-логику: управление заявками, пользователями, отчеты
- **database_module.py** — класс DBManager для подключения к БД и выполнения запросов
- **import_data.py** — скрипт для первичной загрузки данных в базу из CSV файлов
